About this Site

This site is Matt Kanda's and Aaron Anami's website project to track caffeine use using,HTML,PHP and MySQL.

The HTML in this website is used to display online using responsive net, which means that the site loads on different media devices.

PHP and MySQL are used hand in hand to keep the data safe on the database(user names, passwords, entries into tracker). MySQL also allows us to keep the data over time.
